# Web-Project
## Proiect la Tehnici Web 

Realizarea unei aplicatii web pentru un centru de echitatie folosind:
- HTML
- CSS
- JS
- NodeJS
- Express
- EJS

Pentru a rula este necesara instalarea:
- node
- express
- nodemon
- ejs
- body-parser

Pentru a porni serverul pe localhost folosim comanda: 
> nodemon server.js
